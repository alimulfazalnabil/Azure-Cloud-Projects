# IoT Hub Data Simulator

## Overview
A Python client that sends telemetry to Azure IoT Hub.

## Prerequisites
- Azure subscription
- Python 3.10+
- Azure CLI or VS Code Azure extensions

## Getting started
1. Navigate to the code directory.
2. Install requirements with pip install -r requirements.txt.
3. Replace placeholder values such as connection strings and resource names.
4. Deploy the sample to Azure.
